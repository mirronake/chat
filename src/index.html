Cyan Chat

Cyan Chat is an overlay that allows you to show your Twitch chat (and optionally Youtube chat) on screen with OBS, XSplit, and any other streaming software that supports browser sources. It supports your BetterTTV, FrankerFaceZ and 7TV emotes, always at the best available quality. You have many options to customize your chat, like enabling a smooth animation for new messages, or fading old ones after some time.

Commands

Cyan Chat comes with a few mod only commands that you can use in chat.

  • !chat refresh - Refreshes the emotes in chat.
  • !chat reload - Reloads the overlay.
  • !chat test [num] - Sends a number of generated test messages into the chat widget. The number is optional and defaults to 5.
  • !chat rickroll - Rickrolls the chat.
  • !chat tts - Lets you send a tts message (Also can specify a voice with -v {voice}).
  • !chat ytplay [YouTube URL] -d [duration] -s [start_time] - Embeds a YouTube video (duration defaults to 5s, use 0 to play the whole video).
  • !chat ytstop - Removes the currently playing YouTube embed.
  • !chat img [emote_name] or [link] -d [duration] -o [opacity] - Shows the emote image or url image. (duration defaults to 5s if not specified, opacity is 0-1 - default of 1)

  • Force Flag - On the ytplay and img commands if you specify -f it will show it over the chat instead of under.
